Host on GitHub Pages

1

Go to github.com and sign up for a free account. Then click the + icon → New repository.

2

Name it something like americano-scoreboard. Set it to Public. Tick Add a README file. Click Create repository.

3

Inside the repo, click Add file → Upload files. Drag in this HTML file — but rename it to index.html first (GitHub Pages needs this exact name).

4

Scroll down and click Commit changes.

5

Go to Settings → Pages in the left sidebar. Under Source, choose Deploy from a branch, set branch to main, folder to / (root). Click Save.

6

Wait about 60 seconds, refresh the page. Your URL will appear — e.g. https://yourname.github.io/americano-scoreboard. That's your live site — share it with everyone!

Note on scores: Scores save in your browser automatically. The person running the scoreboard just needs to keep the tab open. If you want every phone in the group to see live updates simultaneously, ask about the Firebase version of this app.
