One-time setup

This takes 5 minutes and lets everyone see live scores on their phone. You only do this once — the config saves automatically after.

1

Go to console.firebase.google.com → Add project → give it any name → click through.

2

In the left sidebar: Build → Realtime Database → Create database → pick any region → choose Test mode.

3

Click the gear icon ⚙ → Project settings → scroll to "Your apps" → click the </> Web icon → register app → copy the firebaseConfig block and paste below.
